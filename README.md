DNA Mutation Simulator

A Beginner Bioinformatics Project built using Python.

## Features

- Random DNA mutation
- Mutation position tracking
- Original vs mutated DNA comparison

## Example Output

Original DNA : ATGCGATACG

Mutated DNA : ATGCGATAAG

Position : 9

Change : C -> A

## Technologies Used

- Python
- Random Module
